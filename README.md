# student-result-management-system-java
Java-based Student Result Management System using BST, Stack, Queue, Priority Queue (Heap), and Sorting Algorithms with GUI support.
# 🎓 Student Result Management System

A Java-based Data Structures project designed to manage student records efficiently using multiple DSA concepts.

# 🚀 Features

✔ Add Student Records
✔ Search Student using BST
✔ Display Sorted Students
✔ Show Top 3 Students
✔ Queue & Stack Operations
✔ GUI Interface Included

# 🧠 Data Structures Used

* Arrays
* Binary Search Tree (BST)
* Stack
* Queue
* Priority Queue (Heap)
* Bubble Sort

# 💻 Technologies Used

* Java
* HTML
* Data Structures & Algorithms


# 📌 Functionalities

## 1. Add Student

Allows users to add:

* Student ID
* Name
* Marks

## 2. Search Student

Uses Binary Search Tree (BST) for efficient searching.


## 3. Display Students

Displays students sorted according to marks.


## 4. Top 3 Students

Uses Priority Queue (Heap) to display highest scorers.
